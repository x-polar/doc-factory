# Style Guide — 작성·디자인 표준

AGENTS.md §4(작성 원칙)의 확장판. 슬라이드를 쓸 때 이 기준을 따른다.

## 메시지

- **한 슬라이드, 한 메시지.** 제목이 곧 결론(action title). "매출 추이"(X) →
  "매출은 3년간 2배 성장"(O).
- **So-What.** "그래서?"에 답 못 하면 삭제하거나 병합한다.
- **MECE.** 근거 불릿은 서로 겹치지 않고 빠짐없이.

## 분량

- 슬라이드 제목: 한 줄(약 40자 이내).
- 본문 불릿: 슬라이드당 3~5개, 한 불릿 한 줄.
- 상세 설명·논거·인용은 본문이 아니라 `## notes`(스피커 노트)로.

## 시각

- 색·폰트는 선택한 브랜드의 `theme.yaml`(`brands/<brand>/theme.yaml`) 토큰만 사용.
  임의 색/폰트 금지. 이 덱만의 조정은 `brief.md` frontmatter의 `theme:` 오버라이드로.
- 강조는 색(accent) 하나로. 굵게/색/크기 동시 남발 금지.
- 표·차트 스타일은 `reference/chart-style.md`를 따른다.

## 언어

- 한글·영문 혼용 시 폰트 깨짐 주의. 본문은 브랜드 `theme.yaml`의 `fonts.body`.
- 숫자 단위·표기 일관(예: 30% / 30 % 혼용 금지).
- 용어는 `knowledge/messaging.md`의 승인 용어집을 따른다.

## 출처

- 데이터·인용 슬라이드는 `source:`로 `research/sources.md`의 ID를 단다(추적성).
