# KORI Answers — 단계별 발전 로드맵 (Roadmap)

> 출처: RAG Product Overview 2025.09 (4. 프로젝트 추진 계획). 내용만 정리.

PoC 이후 점진적 고도화 로드맵. 4단계 기능 추가를 통해 MARU 등 사내 IT를 쓰는
대부분 업무를 구성원과 협력 수행하는 **AI 비서 서비스**로 발전시킨다.

> **1단계까지가 본 과제 예상 범위.**

## 1단계 — 개인화 지속 학습 (Continuous Learning)
- 사내 DB·인사정보를 교차 참조하고, 사용자의 일정·데이터 열람 내역으로 현재 업무
  상황을 상시 파악해 맞춤형 응답 제공.
- 사용자별 선호/취향 DB 구축(응답 형식·용어·시각화 스타일 등) → 개인화 DB를 RAG에
  연계해 개인별 최적 답변. 피드백 기반 지속 학습.

## 2단계 — 능동형 업무지원 (Proactive Assistant)
- 단순 질의응답을 넘어 일정·업무 컨텍스트 기반 **선제적 제안**(Time-Triggered).
  예: "오후 2:30 회의 전 관련 보고서 검토 권유", 출근 시 업무 우선순위 안내.
- 업무 흐름 분석으로 사전 알림·문서 추천·작업 가이드. 전 직급에 개인 업무 비서.

## 3단계 — 정밀 분석 지원 (Deep Research)
- ChatGPT Deep Research 유사. 복잡한 질문에 **다단계 추론/Reasoning**으로 정밀
  분석 및 보고서 제공.
- 여러 차례 MARU DB 열람 + 외부 Google 검색 후 검증 과정으로 신뢰도 확보.
- 사용자가 **빠른 응답 vs 정밀 분석**을 명시적으로 선택(기존 서비스 품질 영향 없음).

## 4단계 — 컨텐츠 작성 지원
- 대화형 지시로 업무 문서 자동 작성: 회의록·주간보고·Wiki·출장기록 등.
- 커뮤니케이션 지원: 메일 초안, Knox 기반 일정·미팅 예약. MS Word 보고서 생성/편집.
- 데이터 조회를 넘어 **데이터 활용·창작 단계**로 확장.

## Appendix — Speech-to-Text 회의록 자동 작성 (추가 제안)
- 음성인식(STT) 기반 회의록 자동 작성. 사용자는 녹음 → 자동 생성 회의록을 검토·
  승인해 MARU에 자동 등록. GPU Heavy Server에 Speech Recognition Model 운용.
