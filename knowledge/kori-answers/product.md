# KORI Answers — 제품 개요 (Product)

> 출처: `brands/kori-answers/refs/` (RAG Product Overview 2025.09, KORI Answers
> Product Description 2025.11, Technical Details 2026.03). 디자인은 제외, 내용만 정리.

## 한 줄 정의
기업 내외부 문서 및 DB의 **안전·정확한 검색과 활용**을 위한 **RAG 기반 대화형 AI
서비스**. 사내 폐쇄망(온프레미스)에서 운영되어 데이터 유출이 없다.

- 제품명: **KORI Answers** (내부/구 명칭: **MARU RAG**)
- 1차 적용처: **삼성전자 DS부문 전략마케팅실** — MARU 시스템에 유기적 통합
- 영문 슬로건: "Next Technology for Human Race"

## 3대 핵심 특징
| 축 | 내용 |
|----|------|
| **하이브리드 검색** | 벡터(의미) + 키워드 검색을 결합한 RAG. 결과 내 **출처(In-line citation)** 표기 + 검증 레이어로 신뢰성 확보 |
| **전문가 작성/검수 SQL 조회 도구** | 전문가가 직접 작성·검수한 SQL 쿼리를 LLM/Agent에 **MCP Tool** 형태로 제공 → 정확·효율적 DB 접근. LLM은 SQL을 직접 생성하지 않음 |
| **결정론적 보안 레이어** | 권한에 맞지 않는 데이터는 **처음부터 LLM에 제공하지 않음**. 일관·예측 가능한 보안 |

## 제품 기능
- 사내 **정형/비정형 데이터**를 모두 활용해 사용자 요청에 응답.
- 보고서의 근거가 되는 **참조문서를 항상 제시** → 답변을 원문과 대조 확인 가능.
- 정형 데이터는 가시성 좋은 **표·그래프** 형태로 응답.
- 기획·분석 등 내부 업무의 산출물 작성 효율을 높임.

## 사용 방법
- Perplexity·ChatGPT와 **동일한 대화형 UX**. 별도 학습 없이 누구나 사용.
- 텍스트 입력 후 엔터로 요청 발송, 대화 기록 열람/이동 지원.

## 도입 효과
- **데이터 소스 통합 검색** — DB·문서·뉴스 등 흩어진 정보를 주제별로 한눈에.
- **활용성 높은 결과** — 답변과 함께 근거 문서 제시 → 산출물에 바로 활용.
- **보안/규제 완전 대응** — SSO 연동, 폐쇄망 운영, 외부 데이터 전송 없음.

## 유지관리 · 확장
- 지속적 프롬프트 개선(튜닝·포맷·예외 처리), 사용자 피드백 기반 Q&A 품질 고도화.
- RAG 검색 성능/인덱싱 운영(정확도·랭킹·인덱스 점검), 신규 데이터 소스 편입.
- **기반 LLM 교체** — Llama·Qwen·Gemma·Mistral·GPT-OSS 등 오픈소스 신버전 즉시 반영.

## PoC 결과 (Overview 기준)
- 예상 프롬프트 44종에 대해 **요구사항 만족도 93.2%** (외부 시스템 연계 필요 일부 제외).
- 단답형은 단답형으로, 표가 적합하면 필수 컬럼만 포함하는 등 **응답 형태가 항상 적절**.
