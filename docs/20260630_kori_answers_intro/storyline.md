# Storyline — KORI Answers 소개

> 2단계. 결론과 논리 구조(Pyramid). **여기 승인 후 storyboard 생성.**

## Governing thought (한 줄 결론)
사내 데이터를 **클라우드 없이** 안전하게 AI로 활용할 수 있다 — KORI Answers는
온프레미스 RAG로 **보안과 활용을 동시에** 해결한다.

## 핵심 메시지 (MECE한 근거 축)
1. **문제**: 기업은 사내 지식을 AI로 쓰고 싶지만, 클라우드 LLM은 보안이 걸림돌 (src-01, src-04)
2. **해법**: KORI Answers는 사내 폐쇄망에서 동작하는 RAG AI다 (src-01)
3. **작동 원리(3축)**: 하이브리드 검색·전문가 SQL Tool·결정론적 보안 (src-01, src-02)
4. **입증**: PoC 예상질문 44종 중 93.2% 만족 (src-05)
5. **확장**: 4단계로 AI 비서로 성장 (src-03)

## Horizontal logic 점검 (제목만 읽어도 이야기)
- (표지) KORI Answers — The Sovereign Intelligence Engine
- (문제) 사내 지식을 AI로 쓰고 싶지만, 클라우드는 보안이 걸림돌이다
- (해법) KORI Answers는 클라우드 없이 사내 폐쇄망에서 동작하는 RAG AI다
- (원리1) 하이브리드 검색과 출처 표기로 '신뢰할 수 있는 답'을 만든다
- (원리2) 전문가 SQL Tool로 정형 데이터를 정확·효율적으로 조회한다
- (원리3) 결정론적 보안 레이어가 권한 밖 데이터를 LLM 입력 전에 차단한다
- (입증) PoC에서 예상질문의 93.2%를 만족시켰다
- (확장) 개인화·능동지원·정밀분석·콘텐츠작성으로 AI 비서로 확장된다
- (마무리) No cloud. No compromise.

## So-What 점검
- 각 슬라이드가 governing thought("보안+활용 동시")를 직접 받친다. ✅
