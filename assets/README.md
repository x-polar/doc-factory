# assets/ — 공유 시각 자산

여러 문서가 재사용하는 바이너리 자산. 브랜드가 정해지면 채운다.

```
assets/
├── fonts/    # 폰트 파일(.ttf/.otf) + 라이선스 파일. PDF 변환 환경에도 설치 필요.
├── logos/    # 로고(PNG/SVG). 배경별(밝은/어두운) 버전.
└── icons/    # 아이콘 세트
```

## 규칙
- **폰트 라이선스 파일을 반드시 함께 둔다**(`fonts/<폰트>/LICENSE`). 재배포·임베드
  가능 여부 확인.
- 색·폰트 *값*은 `brand/theme.yaml`이 단일 진실원천. 여기엔 *파일*만 둔다.
- 한글 폰트는 PDF 변환 시 깨짐 방지를 위해 변환 환경(LibreOffice 호스트)에도
  설치되어야 한다(`pdf-export` SKILL 참고).
