# XENOIMPACT 로고 자산

여기에 공식 로고 파일을 둔다. 현재는 **공식 벡터 파일 입수 대기** 상태.

필요한 것(권장):
- `xenoimpact-wordmark-white.svg` (또는 .png, 투명배경) — 다크 배경용 (기본)
- `xenoimpact-wordmark-black.svg` — 밝은 배경용(필요 시)
- `xenoimpact-symbol.svg` — 심볼('X' 셰브론)만 분리된 버전(있으면)
